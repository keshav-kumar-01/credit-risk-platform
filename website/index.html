                Transparency First
                
How Our Explainability Works

                
Three complementary methods ensure every decision is fully understood

            

            

                

                    

                        
01

                        
SHAP Analysis

                    

                    
Based on game theory (Shapley values), SHAP provides mathematically guaranteed fair attribution of each feature's contribution. Every prediction gets a complete breakdown of which factors helped and which hurt.

                    

                        TreeExplainer
                        Force Plots
                        Global/Local
                    

                

                

                    

                        
02

                        
LIME Validation

                    

                    
LIME creates simplified local models around each prediction for model-agnostic validation. When SHAP and LIME agree, confidence in the explanation skyrockets ‚Äî critical for regulatory review.

                    

                        Model Agnostic
                        Local Fidelity
                        Interpretable
                    

                

                

                    

                        
03

                        
Counterfactual "What-If"

                    

                    
Shows applicants the minimum change needed for a different outcome. "If you reduce your loan amount by 20%, you'd be approved." Builds trust and drives repeat applications.

                    

                        Perturbation
                        Actionable
                        Consumer-Friendly
                    

                

            

        

    


    
    

        

            

                Developer API
                
REST API for Production Integration

                
Integrate credit risk assessment into your application in minutes

            

            

                

                    
                    
                    
                

                

                    
import requests

# Full Assessment (50+ fields)
application = {
    "age": 35,
    "credit_amount": 25000,
    "duration": 36,
    "annual_income": 85000,
    "employment_status": "full_time",
    "credit_score": 720,
    "housing_status": "mortgage",
    "loan_purpose": "auto_purchase",
    # ... 50+ fields supported
}

response = requests.post(
    "http://localhost:8000/api/v1/assess",
    json=application,
    headers={"X-API-Key": "demo-key-free-tier"}
)

result = response.json()
print(f"Decision: {result['decision']}")
print(f"Risk Grade: {result['risk_grade']}")
print(f"Probability: {result['probability']:.1%}")
